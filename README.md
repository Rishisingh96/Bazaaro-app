🛒 Bazaaro – Multi-Service E-Commerce Platform

🚀 A modern multi-service e-commerce web application that combines traditional online shopping with on-demand digital services like printing, photo editing, and video editing.

✨ Features
🛍️ E-Commerce Shopping
Browse products (Clothing, Daily Use Items)
Add to cart & place orders
📍 Nearby Shop Ordering
Order from local vendors
Faster delivery system (Blinkit-like concept)
🧾 Service-Based Solutions
📄 Printing & Photocopy
🖼️ Passport Size Photo Creation
🎨 Album & Wedding Card Designing
🎬 Video Editing Services
📤 File Upload System
Upload documents/images for services
Customize service options
🔐 Authentication & Security
Google OAuth Login
OTP Verification (SMTP)
JWT-based Authentication
💳 Online Payment Integration
Razorpay Payment Gateway
📊 Admin Dashboard
Manage Users, Vendors, Products & Services

⚡ Installation & Setup
# Clone the repository
git clone https://github.com/your-username/bazaaro.git

# Navigate to project
cd bazaaro

# Frontend setup
cd client
npm install
npm run dev

# Backend setup
cd server
mvn spring-boot:run
🔐 Environment Variables

Create .env file in both frontend & backend:

# JWT
JWT_SECRET=your_secret_key

# Database
DB_URL=your_mysql_url
DB_USERNAME=root
DB_PASSWORD=your_password

# Razorpay
RAZORPAY_KEY=your_key
RAZORPAY_SECRET=your_secret

# Google OAuth
GOOGLE_CLIENT_ID=your_client_id
GOOGLE_CLIENT_SECRET=your_secret

# SMTP
EMAIL=your_email
EMAIL_PASSWORD=your_password
